# School Hero

School Hero is a Vite-powered K-12 learning game with structured course paths for Math, English Language Arts, Science, and Social Studies.

## Features

- Grade-specific course outlines from Kindergarten through Grade 12
- Practice and challenge modes with grade-aware questions
- Course progress, XP, coins, streaks, hearts, and unlockable skins
- Responsive dashboard and practice UI
- Local progress persistence with `localStorage`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
